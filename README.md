# Seminar-2025-springboot-assignment

이번 한 학기 동안 에브리타임 클론코딩을 주제로 과제를 진행합니다.

3차시 과제에서는 좋아요 기능을 구현하고, 중복 요청이 들어오는 경우의 동시성 문제를 해결합니다.
추가로, 시간이 겹치는 강의를 시간표에 동시에 추가했을 때의 동시성 문제를 해결합니다.

## 시작하기

0차시 과제에서와 같이 앞으로도 데이터베이스로 MySQL을 사용합니다. 로컬 서버를 띄우기 전 `docker compose up -d` 로 MySQL을 띄워주세요.

## 3차시 과제 - 게시글 좋아요 & 동시성 문제 해결

### 게시글 좋아요 기능

- 게시글 좋아요 추가
  - 사용자는 게시글에 좋아요를 누를 수 있습니다.
  - 좋아요는 게시글마다 한 개만 누를 수 있습니다.
  - 좋아요 추가 요청을 여러 번 보내도 에러를 내려주지 않습니다(멱등성).
- 게시글 좋아요 취소
  - 사용자는 자신의 누른 좋아요를 취소할 수 있습니다.
  - 좋아요 취소 요청을 여러 번 보내도 에러를 내려주지 않습니다(멱등성).
- 게시글에 좋아요 수를 포함해서 내려주기
  - 게시글 페이지네이션 조회 또는 단건 상세 조회 시, PostDTO에 좋아요 수를 포함해서 내려줘야 합니다.
